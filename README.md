# Maze Runner 2049

A logic-based 2D maze game developed during my first semester at IIT Patna. This project focuses on implementing smooth character movement, collision detection, and level design using the GameMaker engine.

## 🚀 Overview
The project was an exploration into game logic and state management. It features a top-down perspective where the player must navigate complex environments while avoiding obstacles.

## 🛠️ Technical Stack
* **Engine:** GameMaker Studio 2
* **Language:** GML (GameMaker Language) - C-style scripting
* **Focus Areas:** Collision Physics, Sprite Animation, Logic Flow

## 🧠 Key Engineering Challenges
* **Collision Logic:** Implemented precise tile-based collision to ensure smooth movement without clipping through walls.
* **State Management:** Managed player states (idle, moving, colliding) using efficient conditional logic.
* **Asset Integration:** Handled the pipeline for importing and synchronizing custom sprites and sound effects.

## 📂 Project Structure
* `/objects`: Core game logic and event handling.
* `/scripts`: Reusable functions for movement and UI.
* `/rooms`: Level layouts and environment design.
* `/sprites`: Visual assets and animations.

## 🎮 How to Run
1. Download and install [GameMaker](https://gamemaker.io/en/download).
2. Open the `maze_Game.yyp` file.
3. Press **F5** to run the game.
